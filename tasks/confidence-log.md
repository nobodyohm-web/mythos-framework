## Format
```

**Confidence:** XX/100
**Changes:** N files modified, M files created
**Verification:** ✅/❌ typecheck | ✅/❌ tests | ✅/❌ security
**Concerns:** [any remaining concerns or "None"]
**Accuracy:** [filled in retrospectively — was the confidence accurate?]
```

## Scoring Guide
- 🟢 **90-100**: Ship it. Production-ready. No concerns.
- 🟡 **70-89**: Good work but human review recommended.
- 🟠 **50-69**: Works but has concerns. Flag to user.
- 🔴 **0-49**: Needs human decision. Stop and report.
