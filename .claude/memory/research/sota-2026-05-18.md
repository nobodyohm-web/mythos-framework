# SOTA Ingestion — 2026-05-18

Tier markers: [E] Established · [D] Derived · [C] Conjectured · [S] Speculative.

## 1) Autonomous agent framework architecture
- [E] **File-centric state externalization** (InfiAgent): keep reasoning context bounded by writing persistent state to files rather than holding it in-context.
- [E] **Blackboard pattern**: agents do NOT talk directly; they write to a shared state that an orchestrator reads/dispatches from.
- [E] **Context-Aware MCP / Shared Context Store**: MCP servers coordinate by reading/writing shared context memory.
- [E] **Governance + Observability layer**: Agent OS (policy enforcement), AgentDoG (trajectory diagnostics), AgentGuard (loop detection / anomaly alerts).
- [E] Anthropic "Building Effective Agents" production design patterns are the de-facto baseline.

## 2) Claude Code (hooks · subagents · skills · slash commands)
- [E] Hooks are deterministic — run 100% of the time. PreToolUse matches tool names including Bash, Edit, Write, Read, Glob, Grep, Agent, WebFetch, WebSearch, AskUserQuestion, ExitPlanMode, MCP tool names.
- [E] Subagents run in isolated context windows. Multiple Agent tool calls in a single message execute concurrently.
- [E] **Skills > Commands**: Skills take priority over same-named legacy slash commands. Old `.claude/commands/` is being absorbed into skills.
- [E] Best-practice rule: anything that "must always happen" goes in a hook, not a prompt.

## 3) LLM hallucination / epistemic uncertainty
- [E] **Self-reflection + uncertainty estimation** — agents critique their own outputs using confidence signals.
- [E] **Token-level UQ + think-token injection** (patented 2026): monitor uncertainty per token vs threshold; inject extra computation tokens when uncertainty exceeds bound — operating during streaming.
- [E] **Behaviorally-Calibrated RL**: small models can surpass frontier models on UQ — calibration is a transferable meta-skill decoupled from raw accuracy.
- [E] Calibration-aware metrics treat refusal/abstention as a valid (rewarded) outcome.
- [E] Epistemic uncertainty correlates with hallucination — when forced to generate in low-knowledge regions, models hallucinate.
